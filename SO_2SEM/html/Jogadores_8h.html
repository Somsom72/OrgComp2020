int atualizaPlacar ( int *  placar,
int *  j,
float  fator_inercia,
int  dir 
)

Descarrega buffer do jogador, atualizando o placar para a direção especificada.

Implementa "inércia", ou seja, o placar prefere se mover para a direção em que atualmente aponta. Assume valor de placar em [-100, 100].

Parameters
placarPonteiro para valor do placar a ser atualizado.
jPonteiro para inteiro que representa um jogador (mais precisamente, seu "buffer").
fator_inerciaFloat positivo que indica a intensidade da inérica.
dir-1 caso deseje atualizar para a direção negativa, 1 caso contrário.
Returns
0 caso operação bem-sucedida; 1 caso contrário.
